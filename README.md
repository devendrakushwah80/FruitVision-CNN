# 🍎 Fruits Image Classifier using Deep Learning

## 📌 Project Overview
This project implements a **Fruits Image Classification Model** using Deep Learning.

The model is trained to classify different types of fruits from images using a Convolutional Neural Network (CNN).

This project demonstrates:
- Image preprocessing
- Building a CNN model
- Model training & validation
- Performance evaluation
- Prediction on new images

---

## 🎯 Problem Statement
To build a deep learning model that can accurately classify fruit images into their respective categories.

---

## 🧠 Model Architecture
The model includes:
- Convolutional Layers (Feature Extraction)
- MaxPooling Layers
- Fully Connected (Dense) Layers
- Softmax Output Layer
